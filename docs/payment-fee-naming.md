# 支付费率：单位、字段与命名约定

本文约定平台内**通道 / 商户授权 / 订单快照**三层费率相关字段的中文含义与英文/API 命名对应关系，避免「bps、万分比、费率、手续费」混用。

## 1. 单位（比例一律按「百分数」理解与展示）

| 概念 | API / 代码字段 | 存储与计算 | 界面 |
|------|------------------|------------|------|
| **比例费率** | `*_rate_bps`、`merchant_rate_bps`、`fee_rate_bps` | **万分比整数**：`bps = round(百分数 × 100)`。手续费：`fee_fen = amount_fen * bps / 10000`。 | 按 **百分数（%）** 填写与展示，例如 `0.60` 表示 0.60%，对应 `bps = 60`。 |
| **固定金额** | `fee_fixed_amount`、`upstream_payout_fixed_fee` | 整数，人民币**分**。 | 按「分」填写。 |

**展示用语（全站统一）**

- 比例：`比例费率（%）`；说明文案中可写「百分数」。
- 固定金额：`固定手续费（分）`。

历史字段名仍带 `_bps`：表示库内为整数刻度（万分之），**不要**在界面上要求运营直接填「万分比」数字，应填百分数。

## 2. 计费模式 `fee_mode` / `upstream_payout_fee_mode`

取值 globally 一致：

| 值 | 含义 | 界面短标签 | 完整说明 |
|----|------|------------|----------|
| 1 | 仅比例 | 仅比例 | 仅比例（%） |
| 2 | 仅固定 | 仅固定 | 仅固定（分） |
| 3 | 固定 + 比例 | 固定+比例 | 固定（分）+ 比例（%） |

- **代收订单**：一般仅比例；库中仍可有 `fee_mode` / `fee_fixed_amount` 字段以兼容统一订单结构。
- **代付**：商户授权与上游通道均可使用三种模式（代付通道侧为 `upstream_payout_fee_mode` + 对应比例/固定字段）。

代码常量见前端 `feeSemantics.ts` 中 `FeeMode`。

## 3. 字段分层（谁表示什么）

### 3.1 通道 `channels`（平台相对**上游**的成本）

| 字段 | 含义 |
|------|------|
| `upstream_payin_rate_bps` | 上游代收成本，比例费率（万分比整数 = round(百分数×100)）。 |
| `upstream_payout_rate_bps` | 上游代付成本，同上。 |
| `upstream_payout_fee_mode` | 上游代付计费模式（1/2/3）。 |
| `upstream_payout_fixed_fee` | 上游代付固定手续费（分）。 |

界面前缀统一用：**上游代收 / 上游代付 — …**。

### 3.2 商户产品授权

**代收** `merchant_payin_products`

| 字段 | 含义 |
|------|------|
| `merchant_rate_bps` | 对该商户、该代收产品的**对客**比例费率（万分比整数）。 |

**代付** `merchant_payout_products`

| 字段 | 含义 |
|------|------|
| `fee_mode` | 对客代付计费模式（1/2/3）。 |
| `merchant_rate_bps` | 对客代付比例部分（万分比整数）。 |
| `fee_fixed_amount` | 对客代付固定部分（分）。 |

界面前缀统一用：**对客代收 / 对客代付 — …**。

### 3.3 订单快照 `payin_orders` / `payout_orders`（下单时固化）

| 字段 | 含义 |
|------|------|
| `fee_mode` | 该笔订单采用的计费模式。 |
| `fee_rate_bps` | 与授权侧 `merchant_rate_bps` **同语义**：比例部分万分比整数快照。 |
| `fee_fixed_amount` | 与授权侧 `fee_fixed_amount` **同语义**：固定部分（分）快照。 |
| `fee_amount` / `net_amount` | 实收手续费、净额（分）。 |

命名说明：订单表使用 `fee_rate_bps` 而非 `merchant_rate_bps`，表示**已脱离「商户配置」语境、仅为本单参数快照**；与 `merchant_rate_bps` 数值含义相同。

## 4. 对外产品实体

`payin_products` / `payout_products` 及 `*_product_channels` **不存费率**；费率只在 **通道（上游）** 与 **商户授权（对客）** 两层配置。文案中统一写：「手续费规则在通道与商户授权中配置，不在产品绑定行设置」。

## 5. 与汇率的区别

`fiat_to_usdt_rate`（展示设置）为 **法币↔USDT 汇率**，不是订单手续费 bps，禁止与「比例费率」混称。
